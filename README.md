# Collabers

**Find your project partner. Build something together.**

A platform that connects students to find project partners, hackathon teammates, and collaborators for coursework, startups, and open-source projects.

## Overview

Collabers makes it easy for students to discover projects, connect with like-minded collaborators, and build together. Whether you're looking for a hackathon team, a co-founder for your startup idea, or contributors to your open-source project - Collabers helps you find the right people.

## Features

- **Project Discovery** - Browse projects by category, tech stack, and time commitment
- **Smart Matching** - Find collaborators with complementary skills
- **Real-time Messaging** - Chat with team members and potential collaborators  
- **Application System** - Apply to projects and manage incoming applications
- **Collaboration Tracking** - Monitor active collaborations and project progress
- **Profile System** - Showcase your skills, experience, and portfolio

## Tech Stack

| Layer | Technologies |
|-------|-------------|
| Backend | FastAPI, SQLAlchemy 2.0, PostgreSQL |
| Frontend | React 18, TypeScript, Tailwind CSS, Vite |
| Auth | JWT with refresh tokens, OTP verification |
| Infrastructure | Docker, Nginx |
